What is HTML?

HTML Introduction

  • Standard for hyper text markup lenguaje
  • Describes the structure of a web pages
  • Consist of a series of elements
  • Elements consists on pieces label with content

Basics Elements Explained

  1. La etiqueta "DOCTYPE" define la dleclaracion del documento en cuetion como HTML
  2. La etiqueta HTML es el elemento prrincipal de una pagina HTML
  3. La etiqueta "Title" especifica el titutlo de la pagina web
  4. La etiqueta "Body" especifica el cuerpo de la pagina web
  5. La etiqueta "h1" define un texto de tamaño titular
  6. La etiqueta "h2" define un texto de tamaño subtitular
  7. La etiqueta "p" define un texto de tamaño parrafo

¿Que es un elemento HTML?

Un elemento HTML esta definido por una etiqueta de inicio, seguido de algun contenido y una etiqueta de finalizacion

  • <NombreEtiqueta>: Etiqueta de inicio
  • Contenido: Entre la etiqueta de inicio y la etiqueta de finalizacion
  • <NombreEtiqueta>: Etiqueta de finalizacion

Estilos en HTML

Los estilos en HTML son atributos usados para añadir diseño a las etiquetas HTML, asi como color, tamaño y mas

Estructura de atributos de estilos

Configurar el estilo en un elemento HTML se realiza utlizando el atributo HTML style

  • <tagname style = "property: value;property2: value"> Contenido /tagname>
  • <p style= "color : red">

Elemento HTML "DIV"

La etiqueta <div> es un elemento que toma el ancho de la pagina por defecto y realiza automaticamente un salto de linea despues de cada elemento.<div> se puede interpretar como un contenedor de elementos HTML

Elemento table en HTML

La etiqueta <table> permite colocar los datos en filas y columnas usando sus derivaciones <tr> para definir una fila (table row), <th> para definir los encabezados de la tabla (table header) y <td> para asignar valores a las filas correspondientes a los encabezados de las columnas (table data)
